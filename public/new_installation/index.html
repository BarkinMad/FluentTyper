Quick Start Guide

  • 1. After installation, please reload any open tabs for FluentTyper to activate.
  • 2. Start typing in any text box, and a suggestion menu will appear.
  • 3. Use Arrow Keys (↑, ↓) to select a suggestion.
  • 4. Press Enter or Tab to accept it.
  • 5. Press Escape to hide the suggestions.
